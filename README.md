# Contact Management REST API

Цей проєкт реалізує REST API для керування контактами за допомогою фреймворку **FastAPI**, ORM **SQLAlchemy** та бази даних **PostgreSQL**.

---

## Функціонал
- Повний набір CRUD операцій для контактів (Створення, Читання, Оновлення, Видалення).
- Пошук за іменем, прізвищем або email за допомогою query-параметрів.
- Отримання списку контактів, у яких день народження наступає протягом найближчих 7 днів.
- Автоматична інтерактивна документація Swagger / OpenAPI.

Структура файлів у папці проєкту:

## Інструкція із запуску та тестування

### 1. Запуск бази даних (PostgreSQL)

Переконайтеся, що у вас встановлено та запущено **Docker Desktop**. Запустіть контейнер з БД командою:

```bash
docker run --name some-postgres -p 5432:5432 -e POSTGRES_PASSWORD=mysecretpassword -d postgres
```

---

### 2. Клонування репозиторію та встановлення залежностей

```bash
git clone https://github.com/ErnestSosnovskyi/UMT-pythonweb-hw-08.git
cd UMT-pythonweb-hw-08
# Створення та активація віртуального середовища
python -m venv .venv
# Для Windows:
.venv\Scripts\activate
# Для macOS/Linux:
source .venv/bin/activate
# Встановлення пакетів
python -m pip install -r requirements.txt
```

---

### 3. Запуск вебсервера
**Запустіть додаток за допомогою `uvicorn`:**

```bash
python -m uvicorn main:app --reload
```

Після цього сервер розпочне роботу на адресі `http://127.0.0.1:8000`.

---

### 4. Тестування API через Swagger

1. Відкрийте браузер за адресою: http://127.0.0.1:8000/docs.
2. Ви побачите інтерактивну сторінку з усіма доступними маршрутами.
3. Щоб створити контакт, розгорніть `POST /contacts`, натисніть **Try it out**, заповніть тестове тіло JSON (наприклад, поставте дату народження на кілька днів вперед від сьогоднішньої дати) та натисніть **Execute**.
4. Протестуйте пошук, додавши текст у поле `search` у запиті `GET /contacts`.
5. Перевірте вибірку найближчих святкувань через ендпоінт `GET /contacts/upcoming-birthdays`.
